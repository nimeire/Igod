# 🍔 IGod

Sistema de gerenciamento de cantina para igrejas.

## Objetivo

Facilitar o gerenciamento de pedidos, encomendas e vendas realizadas em cantinas de igreja.

## Funcionalidades

- Cadastro de produtos
- Controle de estoque
- Registro de pedidos
- Encomendas prioritárias
- Relatórios de vendas
- Controle financeiro

## Tecnologias

- Java
- Spring Boot
- MySQL
- HTML
- CSS
- JavaScript
