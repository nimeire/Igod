# 🍽️ iGOD

Sistema de gerenciamento de cantina para igrejas.

O iGOD foi desenvolvido para facilitar o atendimento em cantinas durante cultos e eventos, organizando pedidos, produtos e o fluxo de preparo de forma simples e intuitiva.

> ✅ Primeira versão utilizada com sucesso em produção durante um culto.

---

# ✨ Funcionalidades

- Cadastro de produtos
- Organização por categorias
- Controle de disponibilidade dos produtos
- Cadastro de pedidos
- Geração automática de senha
- Controle de status dos pedidos
- Organização das estações de preparo
- Cadastro de cultos
- Associação de pedidos ao culto ativo
- Registro da forma de pagamento
- Cálculo automático do valor total do pedido

---

# 🛠 Tecnologias

- Java 21
- Spring Boot
- Spring Data JPA
- PostgreSQL
- Hibernate
- Maven
- Lombok

---

# 📱 Objetivo

O objetivo do iGOD é reduzir filas, organizar os pedidos e facilitar o trabalho dos voluntários responsáveis pela cantina da igreja.

O sistema foi pensado para possuir uma interface simples e rápida, permitindo que qualquer voluntário consiga utilizá-lo sem dificuldades.
